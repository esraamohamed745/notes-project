Create and Manage Your All notes in one place

This is a simple and user-friendly note-taking web app. It allows you to create and manage notes with support for different content types.

Features

You can select the type of note from a dropdown menu. Based on the selection, they can enter text, create a list, or upload an image. Once ready, clicking the Add button inserts the note into the main content area.

Each note appears with Edit and Delete buttons, enabling easy updates or removal of content.

The app also includes a search bar to filter notes based on keywords, helping users quickly find what they need.

Additionally, there is a Reviews button that can be used to collect user feedback or display reviews related to the app.
